# 还原App.vue文件计划

## 1. 项目分析

- **项目类型**：Vue 3 + Vite终端博客应用
- **主要依赖**：marked（Markdown解析）、@iarna/toml（TOML配置解析）
- **核心功能**：终端命令处理、文章展示、目录导航、主题设置等

## 2. App.vue文件结构

### 2.1 模板部分
- 终端容器和内容区域
- 欢迎信息和ASCII艺术
- 命令行和输出区域
- 输入框和提示符号
- 响应式设计支持

### 2.2 脚本部分
- 状态管理（对话、命令、当前目录等）
- 命令执行逻辑
- Tab补全功能
- 命令历史记录
- 文章数据加载和处理
- 辅助函数（获取文章信息、补全项等）

### 2.3 样式部分
- 终端样式
- 命令行样式
- 输出样式
- 响应式设计样式

## 3. 核心功能实现

### 3.1 命令处理
- 从commands模块导入命令映射
- 实现命令执行逻辑
- 支持命令参数传递

### 3.2 文章展示
- 加载posts.json数据
- 构建目录结构
- 支持cat命令查看文章内容
- 支持ls命令查看目录
- 支持tree命令查看目录结构

### 3.3 交互功能
- Tab补全（命令和文件名）
- 命令历史记录（上下键切换）
- 自动滚动到底部
- 主题和背景设置

## 4. 实现步骤
