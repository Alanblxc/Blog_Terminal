# 终端博客使用教程

欢迎使用终端博客系统！这是一个基于命令行界面的博客应用，您可以通过简单的终端命令浏览文章和导航目录。

## 基本命令

### 1. 查看帮助信息

```
help
```

显示常用命令列表。如需查看所有命令，请使用：

```
help -l
```

### 2. 列出当前目录内容

```
ls
```
22
显示当前目录下的所有文件和子目录。

### 3. 切换目录

```
cd <目录名>
```

例如，切换到 `articles` 目录：

```
cd articles
```

返回上一级目录：

```
cd ..
```

### 4. 查看文件内容

```
cat <文件名>
```

例如，查看 `example.md` 文件：

```
cat example.md
```

### 5. 查看目录结构

```
tree
```

以树形结构显示当前目录及其子目录的内容。

## 查看文章

### 步骤 1：浏览目录

首先，使用 `ls` 命令查看当前目录下的文章和子目录：

```
ls
```

### 步骤 2：切换到文章目录

如果文章存放在子目录中，使用 `cd` 命令切换到该目录：

```
cd articles
```

### 步骤 3：查看文章内容

使用 `cat` 命令查看文章内容：

```
cat 我的第一篇文章.md
```

## 切换目录示例

假设博客目录结构如下：

```
├── articles/
│   ├── 技术/
│   │   ├── vue3.md
│   │   └── javascript.md
│   └── 生活/
│       ├── 旅行.md
│       └── 美食.md
└── about.md
```

### 示例 1：查看技术文章

```
# 从根目录开始
ls
# 输出：articles  about.md

cd articles
ls
# 输出：技术  生活

cd 技术
ls
# 输出：vue3.md  javascript.md

cat vue3.md
# 显示 vue3.md 的内容
```

### 示例 2：返回根目录并查看 about.md

```
# 从技术目录返回根目录
cd ../..
cat about.md
# 显示 about.md 的内容
```

### 示例 3：使用 tree 命令查看整体结构

```
tree
```

## 常用操作组合

1. **查看所有文章**：
   ```
   tree
   ```

2. **快速浏览文章**：
   ```
   ls articles
   cat articles/example.md
   ```

3. **查找特定文章**：
   ```
   cd articles
   ls
   cat 特定文章.md
   ```

## 提示

- 首次访问网站时，系统会自动执行 `tree` 命令，显示目录结构
- 所有命令均为英文小写，不需要输入引号
- 可以使用方向键上下浏览历史命令
- 如有疑问，随时输入 `help` 查看命令列表

祝您使用愉快！
